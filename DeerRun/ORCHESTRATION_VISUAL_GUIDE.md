## Key Workflows

### Workflow 1: Competitive Ad Analysis (Social Media Primary)

```
Social Media Agent Receives Request:
"Analyze what paid ads our 3 main competitors are running"

├─ Step 1: Access Ad Libraries
│  ├─ Meta Ad Library (Facebook/Instagram ads)
│  ├─ Google Ads Transparency Center
│  ├─ LinkedIn Campaign Monitor
│  └─ TikTok Ads Library
│
├─ Step 2: Collect Data for Each Competitor
│  ├─ Active ads (count, duration)
│  ├─ Ad copy (messaging, angles, CTAs)
│  ├─ Creative assets (images, video formats)
│  ├─ Landing pages (where ads lead)
│  ├─ Estimated budget (based on frequency)
│  └─ Targeting parameters (if visible)
│
├─ Step 3: Competitive Analysis
│  ├─ Ad copy themes & messaging patterns
│  ├─ Product positioning vs. ours
│  ├─ Target audience definitions
│  ├─ Creative style & aesthetic
│  ├─ Testing patterns (variant ad testing)
│  └─ Platform allocation (which platforms get most ads)
│
├─ Step 4: Insights & Recommendations
│  ├─ Key messages competitors emphasize
│  ├─ Gaps in competitor messaging (opportunities for us)
│  ├─ Creative best practices observed
│  ├─ Audience targeting opportunities
│  └─ Recommended ad positioning for us
│
└─ Handoff to Paid Marketing Agent
   "Here's what competitors are doing in paid social;
    here's where we should differentiate"
```

**Output**: Competitive ads analysis report with tactical recommendations

### Workflow 3: New Campaign Competitive Benchmarking

```
Request: "We're planning a paid social campaign for [Product].
         What should we do differently than competitors?"

├─ Step 1: Social Media Agent Analyzes
│  ├─ Current competitor campaigns (all platforms)
│  ├─ Ad copy & creative themes
│  ├─ Targeting & audience sizes
│  ├─ Estimated budgets & duration
│  └─ Performance indicators (engagement, reach)
│
├─ Step 2: Identify Gaps & Opportunities
│  ├─ Messaging angles competitors aren't using
│  ├─ Audience segments underserved
│  ├─ Creative styles we haven't seen
│  ├─ Platform opportunities (if competitors light on specific platform)
│  └─ Unique value props to emphasize
│
├─ Step 3: Deliver Briefing to Paid Marketing Agent
│  ├─ "Competitors emphasize X, Y, Z messaging"
│  ├─ "Audience targets: [segments]"
│  ├─ "Budget level: [range]"
│  ├─ "Creative style: [observations]"
│  └─ "Our differentiation opportunity: [specific angle]"
│
├─ Step 4: Paid Marketing Creates Campaign
│  ├─ Selects differentiated messaging
│  ├─ Targets underserved audiences
│  ├─ Designs creative standing out from competitors
│  └─ Allocates budget strategically
│
└─ Step 5: Launch & Monitor
   ├─ Social Media tracks performance vs. competitors
   ├─ Analytics measures results
   └─ Feedback loop informs optimizations
```

**Output**: Campaign briefing → competitive campaign → performance tracking

### Workflow 4: Content Strategy with Social Insights

```
Request: "Build our social content calendar for next quarter"

├─ Step 1: Content Creator Requests Input
│  ├─ "What are competitors posting?"
│  ├─ "What trends in our category?"
│  ├─ "What's our audience engaging with?"
│  └─ "What should we emphasize?"
│
├─ Step 2: Social Media Agent Provides
│  ├─ Competitor content themes (blog, video, educational)
│  ├─ Trending content formats & topics
│  ├─ Audience engagement patterns
│  ├─ Best posting times & frequency
│  ├─ Hashtag recommendations
│  └─ Influencer partnership opportunities
│
├─ Step 3: Content Creator Builds Calendar
│  ├─ 4 pillars of content (informed by social insights)
│  ├─ Posting schedule optimized for engagement
│  ├─ Hashtag strategy (research-informed)
│  ├─ Influencer collab opportunities identified
│  └─ Unique content angles (differentiated from competitors)
│
└─ Step 4: Social Media Monitors & Optimizes
   ├─ Tracks engagement for each post
   ├─ Identifies top performing content
   ├─ Alerts if trending topics emerge to capitalize on
   └─ Suggests content adjustments based on performance
```

**Output**: Data-driven content calendar → published content → performance feedback

## Daily, Weekly, Monthly Tasks

### Daily (Automated/High Priority)

- [ ] Monitor competitor social posts (all platforms)
- [ ] Track sentiment mentions (brand monitoring)
- [ ] Check ad library for new competitor campaigns
- [ ] Flag any emergencies/urgent trends
- [ ] Review engagement metrics on our accounts (baseline)

**Output**: Daily alert summary to Director

### Weekly (Monday Morning)

- [ ] Comprehensive competitor social analysis
- [ ] Trend identification for category
- [ ] Our brand sentiment summary
- [ ] Influencer activity in our space
- [ ] Recommended strategic responses
- [ ] Ad spend estimation for key competitors

**Output**: Weekly intelligence brief (1-2 pages)

### Monthly (Last Friday of Month)

- [ ] Competitor strategy shift analysis
- [ ] Content performance trends
- [ ] Audience growth/engagement trends
- [ ] Platform shift analysis (where competitors focusing)
- [ ] Market opportunity assessment
- [ ] Forecast for next month's trends

**Output**: Monthly strategic report

### Quarterly (Last week of quarter)

- [ ] Comprehensive competitive positioning update
- [ ] Market share/mindshare assessment
- [ ] Strategic threats & opportunities
- [ ] Recommendation for our positioning refresh
- [ ] Platform strategy recommendations

**Output**: Quarterly strategy briefing

## Success Criteria

### Social Media Agent Success Looks Like:

✅ **Competitive Intelligence**
- Discovers competitor moves 7-10 days before they heavily promote
- Ad analysis provides actionable differentiation angles
- Trends spotted early enough for content team to capitalize

✅ **Content Support**
- Content team's calendar consistently incorporates competitive insights
- Social posts outperform competitors' on similar content
- Trending topics identified 2-3 days before peak

✅ **Paid Campaign Support**
- Paid campaigns have clear differentiation vs. competitors
- Ad copy tested against competitor research
- Budget allocation informed by competitive landscape

✅ **Sentiment & Brand Health**
- Issues identified and escalated within 24 hours
- Sentiment trends visible and tracked monthly
- Crisis avoided through early warning

✅ **Business Impact**
- Campaigns informed by competitive ads outperform
- Content strategy based on social insights drives engagement
- Competitive threats neutralized quickly
