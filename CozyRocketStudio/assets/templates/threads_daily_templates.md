# Threads Daily Templates — CozyRocketStudio

Post one of these every day. Takes 5 minutes.
Rotate through the types to keep the feed varied.

---

## Type 1 — Opinion Take
```
Unpopular opinion:

[YOUR CONTRARIAN VIEW — 1 line]

[WHY — 2-3 short lines]

Agree or disagree?
```

## Type 2 — Micro Lesson
```
Quick AI tip:

[THE TIP — 2-3 lines max]

Full breakdown on Instagram 👆
```

## Type 3 — Question
```
[OPEN QUESTION for your audience]

Real answers only.
```

## Type 4 — Observation
```
I've noticed [PATTERN IN AI / MARKETING / BUSINESS].

[YOUR TAKE ON IT — 1-2 lines]

Is this just me?
```

## Type 5 — Behind the Scenes
```
Building [X] for CozyRocketStudio today.

[WHAT YOU'RE DOING — 1 line]

Here's what I'm learning: [INSIGHT]
```

## Type 6 — Tool Take
```
[TOOL NAME] hot take:

Most people use it for [X].
I use it for [Y].

The difference: [INSIGHT]
```

## Type 7 — Stat or Fact
```
[SURPRISING AI / MARKETING STAT]

What this means for your business:
[YOUR INTERPRETATION — 1-2 lines]
```

## Type 8 — Progress Share
```
Week [X] of building in public.

[ONE HONEST UPDATE — what's working or not]

Following along? More coming.
```
